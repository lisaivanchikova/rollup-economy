## v2 — 6-digit map (built 2026-05-29)

Parallel site at `/6digit/` covers ALL 1,012 US NAICS 2022 6-digit sub-industries with per-vertical agent research:
- Census CBP 2022 firm counts + employment + payroll per code
- V/C/A/M scores derived from web research against verified primary sources (Census, BLS, SEC, CMS, FERC, trade associations) + M&A advisory blogs only for buy/exit multiples
- P&L benchmarks per 6-digit visible on card click (median revenue, EBITDA margin, AI delta)
- Confidence flag (HIGH/MED/LOW) per code so you know whether to trust each row
- 96 specialist agents, one per 3-digit parent, ~3-5 min each
- Honest scoring discipline: no upgrade bias, structurally oligopolistic sub-verticals (Kaiser, DaVita, Class I rails, supermajors) explicitly downgraded
- Verdict distribution: 83 Hell-Yes (8.2%) / 110 Strong / 255 Conditional / 296 Pass / 268 KILL

## What this is

The same methodology — `S_cell = V × C × A × B` — applied to every NAICS 4-digit sub-sector in the US service economy (~80 entries across 13 sectors), with no internal Bobroff Cap context. Profiles come from public industry data only.

## Purpose

Two uses:

1. **Discover blind spots** — sub-sectors that screen Hell-Yes/Strong but we haven't researched yet. Add them to the deep-dive queue.
2. **Validate intuition** — sub-sectors we've already deep-dived should match scores here. Big divergence = either formula or our context is off; both worth checking.

## Caveats

This is a **screen**, not a final verdict.

1. Profile scoring is approximate — single number per sub-sector, real heterogeneity exists within NAICS codes (tier-1 vs tier-2, sub-segment specialty)
2. Workflow modifiers are universal — same modifier applied across all sub-sectors per workflow
3. No founder access scoring — a vertical can be Hell-Yes here but unactionable
4. For any green vertical here, build a focused P&L sheet (like the companion 16-vertical site) before underwriting
